## 📱 TERMUX
<details>
<summary>TAP TO OPEN</summary>

```bash
pkg update && pkg upgrade -y
pkg install git nodejs ffmpeg -y
git clone https://github.com/mrfrankofcc/subzero-md
cd subzero-md
npm install
cp sample.env .env
nano .env
npm run build
npm start
```

</details>

---

🖥️ VPS / LINUX

<details>
<summary>TAP TO OPEN</summary>

```bash
sudo apt update
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs git ffmpeg
git clone https://github.com/mrfrankofcc/subzero-md
cd subzero-md
npm install
cp sample.env .env
nano .env
npm run build
```

RUN WITH PM2

```bash
npm install -g pm2
pm2 start dist/index.js --name subzero
pm2 save
pm2 startup
```

</details>

🐳 DOCKER

<details>
<summary>TAP TO OPEN</summary>

```bash
docker build -t subzero-md .
docker run -d \
  -e SESSION_ID=your_session \
  -e OWNER_NUMBER=your_number \
  -p 5000:5000 \
  subzero-md
```
